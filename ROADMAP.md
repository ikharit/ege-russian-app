## 🔄 Процесс разработки

### Как добавить новую фичу:
1. Git commit перед изменениями (или тег backup)
2. Работать в отдельном файле/компоненте
3. TypeScript проверка: `npx tsc --noEmit`
4. Git commit с описанием
5. Обновить ROADMAP.md

### Как добавить новый статус:
1. Добавить в `src/data/statuses.ts`
2. Условие = ID достижения или 'default'
3. Добавить цвет, emoji, описание
4. Автоматически появится в Profile и Leaderboard

### Как добавить новое достижение:
1. Добавить в `src/data/achievements.ts`
2. Добавить проверку в `progressStore.ts::checkAchievements()`
3. Добавить иконку в `src/data/achievementIcons.ts`
4. Добавить progress в `getAchievementProgress()`
