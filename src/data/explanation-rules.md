# Правила для объяснений в заданиях ЕГЭ

## Задание 9 — Пропущенные буквы (корни, приставки, иноязычные)

### Шаблоны объяснений по типам:

#### 1. Проверяемые корни (чередование гласных)
**Формат**: "Корень -[корень]- (чередуется с -[альтернатива]-). Проверочное слово: [слово]."

**Примеры**:
- пловец: "Корень -плав- (чередование а/о). Проверочное слово: плавать."
- ростовщик: "Корень -рос- (чередуется с -раст-). Проверочное слово: росток."
- выскочка: "Корень -скоч- (чередуется с -скак-). Проверочное слово: скакать."

#### 2. Проверяемые корни (чередование согласных)
**Формат**: "Корень -[корень]- (чередуется с -[альтернатива]-). Проверочное слово: [слово]."

**Примеры**:
- выдирать: "Корень -дир- (чередуется с -дра-). Проверочное слово: драть."
- обжигаться: "Корень -жиг- (чередуется с -жег-). Суффикс -а-, поэтому пишется -и-."

#### 3. Непроверяемые (иноязычные, словарные)
**Формат**: "Иноязычное слово / Словарное слово. Непроверяемая гласная [буква]."

**Примеры**:
- дирижёр: "Иноязычное слово (франц. diriger). Непроверяемая гласная и."
- мораторий: "Иноязычное слово (лат. moratorius). Непроверяемая гласная о."
- метафора: "Иноязычное слово (греч. metaphora). Непроверяемая гласная о."

#### 4. Приставки
**Формат**: "Приставка [приставка]- + корень -[корень]-."

**Примеры**:
- старожилы: "Приставка старо- + корень -жил- (от глагола жить)."
- исхоженный: "Приставка ис- + корень -ход-."

#### 5. Суффиксы (задание 10)
**Формат**: "Суффикс -[суффикс]-: [правило]."

**Примеры**:
- щеголеватый: "Суффикс -еват- (прилагательные с оттенком неполноты признака)."
- находчивый: "Суффикс -ив- (прилагательные от глаголов несовершенного вида)."

## Задание 13 — НЕ с разными частями речи

### Шаблоны:

#### 1. НЕ с прилагательными
**Формат**: "НЕ с прилагательным пишется [слитно/раздельно]: [правило]."

**Правила**:
- Слитно: если нет противопоставления (неумолимый)
- Раздельно: если есть противопоставление с союзом "а" (не умолимый, а...)

#### 2. НЕ с наречиями
**Формат**: "НЕ с наречием пишется [слитно/раздельно]."

**Правила**:
- Наречия на -о/-е: слитно (невежливо)
- Наречия с противопоставлением: раздельно

#### 3. НЕ с деепричастиями
**Формат**: "НЕ с деепричием пишется [слитно/раздельно]."

**Правила**:
- Слитно: без противопоставления
- Раздельно: с противопоставлением или наречие перед деепричастием
