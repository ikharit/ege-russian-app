# Анализ atomization-project / raw_words.txt

## Общая информация

- **Файл**: `data/raw_words.txt`
- **Строк**: 1274
- **Формат**: слова с пропусками (`..`), иногда с пояснением в скобках
- **Назначение**: сырой материал для заданий 9–10 ЕГЭ (приставки, корни, суффиксы)

---

## Выявленные атомарные категории

### 1. Приставки ПРЕ- / ПРИ-

**Подтипы:**
- `ПРЕ-` словарные (нельзя проверить): преисполнен, премудрый, преследование, претворить, преграда, премадонна, преамурский
- `ПРИ-` словарные: приобрёл, приозёрский, приоткрыть, привязанный, приклеить
- `ПРЕД-` словарные: предзащита, предзаказ, предъюбилейный
- `ПОД-` проверяемые: подкараулить, подножие, подправить

**Атом**: `prefix_pre_pri` с податомами:
- `pre_pri_dictionary` — словарные ПРЕ/ПРИ
- `pre_pri_verifiable` — проверяемые (приставка + проверочное слово)

### 2. Приставки ВС- / ВЗ-

- `в..колосилась` → ВСКОЛОСИЛАСЬ (вс- перед к)
- `вз..браться` → ВЗОБРАТЬСЯ (вз- перед б)
- `в..пыхнуть` → ВСПЫХНУТЬ
- `в..езд` → ВЪЕЗД

**Атом**: `prefix_vs_vz` с податомами:
- `vs_before_k` — ВС- перед к, х, п (всколоситься, вспыхнуть)
- `vz_before_b` — ВЗ- перед б, д, з (взобраться, вздремнуть)
- `v_with_hard_sign` — въезд, съезд (разделительный Ъ)

### 3. Приставки ЧЕРЕС- / ЧРЕЗ-

- `чере..чур` → ЧЕРЕСЧУР
- `чере..полосица` → ЧЕРЕСПОЛОСИЦА
- `чре..вычайное` → ЧРЕЗВЫЧАЙНОЕ

**Атом**: `prefix_cheres_chrez` — словарные слова

### 4. Приставки РАС- / РАЗ- / РОЗ-

- `ра..каяние` → РАСКАЯНИЕ (РАС- перед к)
- `раз..брать` → РАЗОБРАТЬ (РАЗ- перед глухими)
- `ро..черк` → РОСЧЕРК (РОС- перед ч)
- `роз..ск` → РОЗЫСК (РОЗ- перед з)

**Атом**: `prefix_ras_raz` с податомами:
- `ras_before_k_h` — РАС- перед к, ч, п (раскаяние, расширяться)
- `raz_before_deaf` — РАЗ- перед глухими (разобрать, разъехаться)
- `ros_roz_special` — особые случаи (росчерк, розыск)

### 5. Приставки БЕЗ- / БЕС-

- `без..скусный` → БЕЗЫСКУСНЫЙ
- `бес..правный` → БЕСПРАВНЫЙ
- `бес..конечный` → БЕСКОНЕЧНЫЙ

**Атом**: `prefix_bez_bes` — глухая/звонкая согласная

### 6. Приставки ИЗ- / ИС- / ИЗО- / ИСПО-

- `изра..ходовать` → ИЗРАСХОДОВАТЬ (ИЗ- перед р)
- `ис..подтишка` → ИСПОДТИШКА (ИС- перед п)
- `ис..следовать` → ИССЛЕДОВАТЬ (ИС- перед с)

**Атом**: `prefix_iz_is` с податомами:
- `iz_before_r` — ИЗ- перед р
- `is_before_p_s` — ИС- перед п, с

### 7. Приставки С- / СО- / СУБ-

- `с..знова` → СЫЗНОВА (С- + Ы)
- `с..грать` → СЫГРАТЬ
- `суб..ективный` → СУБЪЕКТИВНЫЙ

**Атом**: `prefix_s_so` с податомами:
- `s_with_y` — СЫ- (сызнова, сыграть, съесть)
- `sub_with_hard_sign` — СУБЪ- (разделительный Ъ)

### 8. Приставки ОБ- / ОБЕЗ- / ОБЕС-

- `обез..яний` → ОБЕЗЬЯНИЙ (ОБЕЗ- + Ь)
- `обе..цветить` → ОБЕСЦВЕТИТЬ (ОБЕС- перед ц)
- `об..ективный` → ОБЪЕКТИВНЫЙ (ОБЪ- + разделительный Ъ)

**Атом**: `prefix_ob_obez` с податомами:
- `obez_with_soft_sign` — ОБЕЗЬ- (обезьяний)
- `obes_before_ts` — ОБЕС- перед ц
- `ob_with_hard_sign` — ОБЪ- (объективный)

### 9. Приставки НЕ- / НИ-

- `не..гибаемый` → НЕГИБАЕМЫЙ (НЕ-)
- `ни..ходить` → НИСХОДИТЬ (НИ-)
- `непр..будный` → НЕПРИБУДНЫЙ

**Атом**: `prefix_ne_ni` с податомами:
- `ne_with_noun` — НЕ- с существительными/прилагательными
- `ni_with_verb` — НИ- с глаголами (нисходить)

### 10. Чередование в корне

- `б..ющийся` → БЬЮЩИЙСЯ (бить)
- `л..ющийся` → ЛЬЮЩИЙСЯ (лить)
- `в..ющееся` → ВЬЮЩЕЕСЯ (вить)

**Атом**: `root_alternation` с податомами:
- `root_b_bj` — бить → бью
- `root_l_lj` — лить → лью
- `root_v_vj` — вить → вью

### 11. Другое (редкие приставки)

- `до..` → до- (догнать, добела, дословно, досыта)
- `на..` → на-/над- (насчёт, наконец, надстроить)
- `от..` → от- (отыграться)
- `за..` → за-/зас- (закаляющийся, заинтересованный)
- `во..` → во-/вос- (вопевать, воспламениться)
- `пре..` → пре- (преграда, премадонна)
- `при..` → при- (приамурский)
- `ро..` → рос-/роз- (росчерк, розыск)
- `трёх..` → трёхъ- (трёхъядерный — разделительный Ъ)
- `меж..` → меж-/между- (межъязыковой)

## Предлагаемая структура данных для атомизации

```typescript
// src/data/atomization.ts

export interface Atom {
  id: string           // 'prefix_pre_pri'
  name: string         // 'Приставки ПРЕ-/ПРИ-'
  description: string  // 'Правописание приставок пре-/при-...'
  taskNumbers: number[] // [9, 10] — в каких заданиях ЕГЭ встречается
  parentAtom?: string  // id родительского атома (иерархия)
  difficulty: 'easy' | 'medium' | 'hard'
}

export interface AtomizedWord {
  word: string         // 'преисполнен'
  rawForm: string      // 'пр..исполнен (отваги)'
  atoms: string[]      // ['prefix_pre_pri', 'pre_pri_dictionary']
  taskNumber: number   // 9
  difficulty: 'easy' | 'medium' | 'hard'
  explanation: string  // 'Словарное слово с приставкой ПРЕ-...'
  rule: string         // 'Перед и пишется ПРЕ- в словарных словах...'
}

export interface UserAtomProgress {
  atomId: string
  totalAttempts: number
  correctCount: number
  accuracy: number     // 0–100
  lastAttemptAt: string
  masteryLevel: 'new' | 'learning' | 'review' | 'mastered'
}
```

## Связь с ege-russian-app

### Как использовать атомизацию:

1. **Диагностика ошибок**
   - Ученик ответил неправильно на `пр..исполнен`
   - Система определяет атом: `prefix_pre_pri` → `pre_pri_dictionary`
   - В статистике: "Слабое место: словарные слова с ПРЕ-"

2. **Адаптивная практика**
   - Если accuracy по `pre_pri_dictionary` < 70%
   - Система предлагает больше вопросов именно с этим атомом

3. **Teacher-панель**
   - Вместо "Слабые темы: Задание 9"
   - Показывать: "Слабые атомы: ПРЕ-/ПРИ- словарные (65%), ВС-/ВЗ- (58%)"

4. **Прогноз балла**
   - Атомы с accuracy < 50% → риск потери баллов
   - Атомы с accuracy > 90% → надёжный балл
