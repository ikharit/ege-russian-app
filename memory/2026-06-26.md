## Дополнение: Data audit fixes (Agent 2) — 5 items (2026-06-26, 19:00)

### 1. RAG warnings — fixed false positive
- **Проблема**: `verify-rag.js` искал "проверяемый" и "непроверяемый" через `.includes()`. Но "непроверяемый" содержит "проверяемый" как substring — это давало 268 false positive warnings.
- **Решение**: Добавлена функция `hasStandaloneWord()` — для пары "проверяемый"/"непроверяемый" проверяет, что "проверяемый" не является частью "непроверяемый".
- **Файл**: `scripts/verify-rag.js` (14 строк изменений)
- **Результат**: `npm run validate:rag` → 1379 entries, 0 errors, 0 warnings ✅

### 2. Line endings — AGENT_TASKS.md CRLF → LF
- **Проблема**: `AGENT_TASKS.md` был в CRLF (`\r\n`), все остальные файлы — LF. Это мешало инструментам редактирования.
- **Решение**: `sed -i 's/\r$//' AGENT_TASKS.md`
- **Результат**: `file AGENT_TASKS.md` → "UTF-8 text" (без CRLF)

### 3. Миграции Supabase — дедупликация + GIN индексы
- **Проблема**: `001_unified_tracking.sql` и `001_unified_tracking_v2.sql` — оба создавали одни и те же таблицы (answer_logs, teacher_student_links, student_word_errors, триггер, функция). v1 — ALTER TABLE (legacy), v2 — CREATE TABLE (baseline для fresh проектов).
- **Решение**: `001_unified_tracking.sql` перемещён в `supabase/migrations/archived/` (legacy). Создан `003_gin_indexes.sql`:
  - `idx_user_progress_stats` (GIN on user_stats)
  - `idx_user_progress_exam` (GIN on exam_results)
  - `idx_user_analytics_behavior` (GIN on behavior_profile)
  - `idx_user_analytics_snapshots` (GIN on daily_snapshots)
  - `idx_user_progress_updated` / `idx_user_analytics_updated` (B-tree on updated_at)
- **Файлы**: `supabase/migrations/archived/001_unified_tracking.sql`, `supabase/migrations/003_gin_indexes.sql`

### 4. Explicit knowledge graph — graph-relations.json
- **Проблема**: Графовые связи существовали, но были неявными (relatedAtoms, questionMapping, canonicalWordId). Нельзя было query по графу.
- **Решение**: Создан `scripts/build-graph-relations.js` — парсит `knowledge-index.json`, `questionMapping.ts`, `atoms.ts` и генерирует `public/data/graph-relations.json`:
  - Nodes: 1460 (15 tasks, 1209 words, 89 rules, 101 atoms, 46 questions)
  - Edges: 4954 (word→atom, word→task, rule→task, rule→atom, atom→parent, word→question)
- **Скрипт добавлен в build**: `package.json` → `"build:graph": "node scripts/build-graph-relations.js"` → `"build": "npm run build:rag && npm run build:graph && vite build"`
- **Файлы**: `scripts/build-graph-relations.js`, `public/data/graph-relations.json` (33,384 строки)
