## Формат обмена данными

### Excel-файл (Chат 1 → Chат 2)
Создаётся в: `data/atomization/task10/output/task10_atomized.xlsx`

Столбцы:
1. `id` — уникальный ID
2. `word_raw` — слово с пропуском (как в исходнике)
3. `context` — контекст в скобках
4. `correct_letter` — правильная буква
5. `full_word` — полное слово
6. `main_category` — основная категория (ПРЕ-/ПРИ-, З-/С-, Ы-/И-, Ъ-/Ь-, РА-/РО-, ПРА-/ПРО-, Неизменяемые)
7. `subcategory` — подкатегория атомизации
8. `rule` — правило/пояснение
9. `cluster` — кластер (созвучные/однокоренные слова)
10. `source` — источник (Дощинский-2026 / Дощинский-2025)
11. `difficulty` — сложность (1-5, опционально)

### JSON-файл (Chат 1 → Chат 2, опционально)

- Машиночитаемая структура для импорта в БД

## Правила ЕГЭ для задания №10 (кратко)

### ПРЕ-/ПРИ-
- ПРЕ- = «до, перед, очень» (представить, превысить, превосходный)
- ПРИ- = «присоединение, близость, слабость» (привязать, прийти, прибежать)
- Словарные: президент, премьера, премадонна, премудрый, прескорбный, преизбыточный — всё ПРЕ-

### З-/С-
- Изменяемые: без-/бес-, воз-/вос-/вз-/вс-, из-/ис-, низ-/нис-, раз-/рас-/роз-/рос-, чрез-/чрес-/через-/черес-
- З перед гласными и звонкими (б, в, г, д, ж, з, й, л, м, н, р)
- С перед глухими (к, п, т, с, ф, х, ц, ч, ш, щ)
- Неизменяемая: с- (сделать, сбежать) — нет приставки з-

### Ы-/И-
- Ы после русских приставок (вз-, с-, раз-, под-, без- и т.д.)
- И после иноязычных приставок (суб-, контр-, транс-, супер-, пан-, пост-, дез-, дис-, интер-, архи-, гипер-, гипо-, анти-, инфра-, нео-, ультра-, экс-, экстра-, меж-, сверх-)
- Исключения: взимать (И), меж- (И), сверх- (И)
- В корнях без приставки: пединститут, мединститут, спортинвентарь — И

### Ъ-/Ь-
- Ъ — разделительный после приставки перед е, ё, ю, я (подъезд, съезд, объявить, разъяснить)
- Ь — в корне (дьяк, пьеса, вьюга, бурьян, мышьяк)
- Ь — в заимствованиях (барельеф, интервью, конъюнктивит, адъютант, фельдъегерь, павильон, шампиньон)
- Ъ в сложных словах с числительными: двухъярусный (двух + ярусный), трёхъядерный, трёхъязычный

### РА-/РО- / РАЗ-/РОЗ- / РАС-/РОС-
- РА- под ударением (расказать), РО- без ударения (роспись, росчерк)
- РАЗ-/РАС- без ударения, РОЗ-/РОС- под ударением
- Двойные согласные: рассчитать (рас- + с-), расчёт (ра- + счёт)

### ПРА- / ПРО-
- ПРА- = «древний, родство» (прадед, праязык, прародина, праславянский, праотцы)
- ПРО- = «прототип, действие в интересах» (прообраз, пророссийский, противо-)

### Неизменяемые приставки
- с-, со-, под-, подо-, подъ-, над-, надо-, от-, ото-, об-, обо-, в-, во-, вы-, до-, за-, на-, не-, недо-, о-, пере-, по-, пред-, предо-, преди-, предъ-, про-, разо-, у-, без-, вне-, и др.
