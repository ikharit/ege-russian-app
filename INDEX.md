## Структура папок

```
src/
├── pages/           # 22 страницы (React-компоненты, по одной на экран)
├── components/      # 15+ компонентов (QuestionCard, Hearts, Popover, TheoryViewer...)
├── stores/          # 7 Zustand stores (по одному на модуль + progressStore)
├── data/
│   ├── sections/    # 17 разделов курса (задания 1–26 в виде уроков)
│   ├── theory/      # Теория по заданиям (task4, 9, 10, 11, 12, 14...)
│   ├── gsheets/     # Данные учеников из Google Sheets (9 человек)
│   ├── atomization/ # Атомы (микро-навыки)
│   └── task*N*Questions.ts  # Данные для тренажёров (5, 10, 16)
├── lib/             # Утилиты (sounds.ts, theoryMapper.ts...)
└── types/           # TypeScript interfaces (index.ts)
```

## ⚠️ Известные проблемы (не трогать без согласования)

### 1. Line endings (CRLF/LF)
~10 файлов показываются как `modified` в `git status`, но `git diff` пустой. Это разница CRLF ↔ LF. **Не трогать** — иначе будет mass-reformat, конфликты с другими агентами.

### 2. Supabase (закомментирован в App.tsx)
Настроен auth + sync, но закомментирован. Не удалять — может быть активирован позже.

### 3. Task16Trainer.tsx — legacy
Файл убран из UI (нет роута в App.tsx, нет карточки в Dashboard), но оставлен в файловой системе. Задание 16 теперь в `punctuation.ts` как уроки. Если нужно вернуть тренажёр — раскомментировать в App.tsx и Dashboard.tsx.
